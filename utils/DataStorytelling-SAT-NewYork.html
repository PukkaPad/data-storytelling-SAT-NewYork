def load_files(path):
    """
    Takes csv files, convert them to data dictionary.
    Args:
        Data path
    Returns:
        data dictionary.
    """
    data = {}
    for file_ in glob.glob(path):
        file_name = os.path.splitext(os.path.basename(file_))[0]
        d = pd.read_csv("{}".format(file_))
        data[file_.replace(file_, file_name)] = d
    return data

Inner join produces only the set of records that match in both Table A and Table B.

Outer join produces the set of all records in Table A and Table B, with matching records from both sides where available. If there is no match, the missing side will contain null.
